Linux Mint

Version 17 LTS Cinnamon

Codename "Qiana" is the latest version of Linux Mint and since it is LTS it is supported until April 2019. The Cinnamon version is the version you want if u are all about the latest in technology since it uses GNOME 3. *GNOME 3.x aims to provide a single interface for desktop computers and devices such as smart phones and tablet computers.

Version 17 LTS Mate

The Mate version is the version you want if your computer is a bit outdated, and would run better since it uses GNOME 2.

Info

Download 64-bit version if computer have more than 4GB RAM, 32-bit if less than 4GB RAM.
If you are familiar with sudo code, there is loads of things to do to make the system more personal.
Multimedia support: Yes, VLC, Brasero, Banshee.
Office programs: Yes, LibreOffice.
Web-browser: Yes, Firefox.
E-mail application: Yes, Thunderbird Mail.
Software Manager: Yes, place to install new software.
