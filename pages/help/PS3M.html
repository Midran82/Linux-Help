PS3 Media Server

Info

*PS3 Media Server is a DLNA-compliant UPnP media server. *wikipedia
With this program you can stream your mediafiles to your preferred mediacenter.
This works with Ubuntu Desktop/Server and Linux Mint (might also work on other Linux OS).

Installation

Step 1: Open your terminal window
Step 2: sudo add-apt-repository ppa:happy-neko/ps3mediaserver
Step 3: sudo apt-get update
Step 4: sudo apt-get install ps3mediaserver
After this is done you can start the program and set it up to stream your media (even the NAS folder if you used my NAS guide).
