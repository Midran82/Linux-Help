Minecraft Server

Info

Here I will show you how to host your own Minecraft Server on your Linux system.
This work with Ubuntu Desktop/Server and Linux Mint (might also work on other Linux OS).

Installation

Step 1: open Terminal window
Step 2: sudo apt-add-repository ppa:webupd8team/java
Step 3: sudo apt-get update
Step 4: sudo apt-get install oracle-java7-installer
Step 5: mkdir minecraft : cd minecraft
Step 6: Download minecraft server file:
wget -O minecraft_server.jar https://s3.amazonaws.com/Minecraft.Download/versions/1.7.9/minecraft_server.1.7.9.jar
Step 7: java -Xms1536M -Xmx1536M -jar minecraft_server.jar nogui -starting the Minecraft Server : "stop" to stop the server
