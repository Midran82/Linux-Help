OS Installation


OS to USB

This will show you how to install the OS onto your USB or SD-Card for installation to your computer.
Step 1: First you need is to download the OS you want to install into the USB/Card. Some you can find download links to on this page.
Step 2: Next you need to download a program that helps you install the OS. That program would be something like this: USB Installer that i use.
Step 3: Install the operation system and your off to go.

NOOBS Installation

To install NOOBS on SD-Card for Raspberry pi is a little different.
Step 1: You need to format the SD-Card before you copy NOOBS into it. This you can do with SD Formatter.
Step 2: Double check when using program that you are formatting the SD-Card before starting and then format it.
Step 3: Download NOOBS and then extract the files onto a temporary folder.
Step 4: Drag the files extracted into your SD-Card Drive and safely remove your SD-Card and put it into the Raspberry Pi and boot it up.
Step 5: Install preferred operation system when prompted and after it's done you are ready to go.
